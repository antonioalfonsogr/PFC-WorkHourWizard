Proyecto WorkHourWizard

Esta aplicación web es mi proyecto final del ciclo formativo de Desarrollo de Aplicaciones Web (DAW). Consiste en una herramienta para gestionar las horas de trabajo de los empleados dentro de los horarios que ellos seleccionen como disponibles. El proyecto ha sido desarrollado utilizando Spring Boot, Angular, MySQL y Docker. Durante el desarrollo, he aplicado diversos conceptos y técnicas aprendidas en el ciclo formativo, como el diseño de interfaces de usuario, el manejo de bases de datos y la programación orientada a objetos. Además, he utilizado Docker para el despliegue de la aplicación, lo que ha facilitado la configuración y la gestión del entorno de ejecución.
